# FlixFree - Watch Free Movies Online

A beautiful, fully functional **free movie streaming website** built with HTML, Tailwind CSS, and vanilla JavaScript.

![FlixFree Preview](https://picsum.photos/id/1015/800/400)

## ✨ Features

- **Netflix-style dark UI** with smooth animations
- **Responsive design** - works on mobile, tablet, and desktop
- **Hero banner** with featured movie
- **Trending & Popular** movie sections
- **Search functionality** (type and press Enter)
- **Genre filters** (Action, Sci-Fi, Drama, Comedy)
- **Movie details modal** with play button
- **Fake video player** (demo mode)
- **Hover effects** and keyboard shortcuts

## 🚀 How to Run Locally

1. Download or clone this repository
2. Open `index.html` in your browser
3. Or run a local server:

```bash
# Using Python
python3 -m http.server 8080

# Then visit http://localhost:8080
```

## 🌐 Live Demo

**Coming soon** - Deploy it to GitHub Pages, Netlify, or Vercel in under 2 minutes!

### Deploy to GitHub Pages:
1. Create a new public repo on GitHub
2. Upload `index.html` and `README.md`
3. Go to **Settings > Pages > Deploy from main branch**
4. Your site will be live at: `https://yourusername.github.io/repo-name`
